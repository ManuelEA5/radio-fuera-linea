# 📋 CONTRIBUIR AL PROYECTO

¡Gracias por querer contribuir a Radio Fuera de Línea!

## Cómo reportar errores

1. Ve a **Issues** en GitHub
2. Haz clic en **New Issue**
3. Describe el problema con:
   - Qué estabas haciendo
   - Qué pasó
   - Qué esperabas que pasase
   - Logs de error (si es posible)

## Cómo sugerir mejoras

1. Abre un **Discussion** o **Issue**
2. Etiqueta como `enhancement`
3. Describe la mejora y por qué sería útil

## Cómo hacer cambios

1. **Fork** el repositorio
2. **Clone** tu fork: `git clone https://github.com/TU_USUARIO/radio-fuera-linea.git`
3. **Crea rama**: `git checkout -b feature/mi-mejora`
4. **Haz cambios**
5. **Commit**: `git commit -m "Descripción clara del cambio"`
6. **Push**: `git push origin feature/mi-mejora`
7. **Pull Request**: Describe tus cambios

## Estándares

- Código limpio y documentado
- Comentarios en español
- Pruebas locales antes de PR
- Actualizar documentación si es necesario

## Áreas donde podemos mejorar

- 🌍 Traducción a otros idiomas
- 📡 Integración con más modos radio
- 🎨 Mejoras de interfaz
- 📚 Documentación y tutoriales
- 🐛 Correcciones de bugs
- 🚀 Nuevas características
